# AI-Website-Summarizer-Ollama
# LLM Website Summarizer

A Python application that:

- Scrapes webpage content using BeautifulSoup
- Uses local Ollama models via OpenAI-compatible API
- Generates concise summaries of websites

## Technologies

- Python
- BeautifulSoup4
- Requests
- Ollama
- OpenAI SDK



Output:
A concise AI-generated summary of the website content.
